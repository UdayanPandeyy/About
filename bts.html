Bug Tracking System

The project aims at providing ease on customer's end and fixing their software bugs. It's a type of issue tracking System. It allow end-users to add their bugs and admin acts as a middleware between customer and engineers.The system allows a group of engineers or individual engineer to keep track of fixed as well as unfixed bugs in their software.

Language:

Spring,Hibernate

Database:

MySQL

User Interface Design:

HTML,CSS, JQUERY AJAX, JAVASCRIPT,Angular 6.

SOFTWARE:

XAMP SERVER.

DURETION:

ONE MONTH
